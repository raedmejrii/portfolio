# raedmejri.dev — Personal Portfolio

Built with React, Vite, and TypeScript. Deployed on Vercel.

🌐 Live: [raedmejri.dev](https://raedmejri.dev)

## Stack
- React 18 + TypeScript
- Vite
- EmailJS (contact form)
- Vercel (CI/CD)

## Features
- Dark / Light mode
- Typewriter effect
- Animated counters
- Contact form
- Download CV
- Fully responsive
